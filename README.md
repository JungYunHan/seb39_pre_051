## 기술 스택

### FrontEnd

- `HTML`
- `CSS`
- `Javascript`
- `React`
- `Redux Toolkit`
- `Styled Components`

### **BackEnd**

- `Java 11`
- `Gradle 7.4.2`
- `Spring Boot 2.7.3`
- `H2`
- `MySQL`
- `Spring Data JPA`
- `Spring Security`
- `JWT`
- `Spring Validation`
- `Lombok`
- `Mapstruct`
- `Swagger`

### **문서/협업**

- `Github Projects`
  - `Issue`
  - `Milestones`
- `Git`
- `Slack`
- `Zoom`

### 브랜치 전략

- `git flow`
  
  - main : 운영환경에서 사용되는 브랜치입니다.
    
  - dev: 개발환경에서 사용되는 브랜치입니다.
    
  - feat: 기능을 개발할 때 사용되는 브랜치입니다.
